# API Documentation

Base URL (default): `http://localhost:8000/api/v1`

Interactive OpenAPI docs are auto-generated by FastAPI at
`http://localhost:8000/docs` (Swagger UI) and `/redoc` once the server
is running.

## Health

### `GET /`
Root health check.

### `GET /api/v1/health`
Detailed health check, includes whether the YAMNet model has been
loaded yet (it loads lazily on first analysis request).

**Response**
```json
{"status": "ok", "app_name": "NoiseGuard AI", "version": "1.0.0", "model_loaded": true}
```

---

## Analysis

### `POST /api/v1/analysis`
Upload an audio file and run the full AI pipeline (feature extraction
+ YAMNet classification + loudness estimation + interpretation).

**Request**: `multipart/form-data`, field `file` (wav/mp3/m4a/ogg/flac/webm, max 25MB).

**Response** `200 OK`
```json
{
  "id": "b3f1...",
  "filename": "courtyard.wav",
  "created_at": "2026-07-18T12:00:00Z",
  "predictions": [{"label": "Jackhammer", "confidence": 0.74}],
  "top_prediction": {"label": "Jackhammer", "confidence": 0.74},
  "features": {"duration_sec": 6.4, "rms_mean": 0.19, "...": "..."},
  "loudness": {"dbfs": -14.3, "category": "loud", "explanation": "..."},
  "environmental_interpretation": "Detected class: 'Jackhammer'. ...",
  "recommendations": ["Verify the construction site's permitted working hours...", "..."],
  "waveform_preview": [0.01, 0.02, "..."]
}
```

**Errors**: `400` unsupported extension / too large / empty / too short; `500` unexpected pipeline error.

## History & Statistics

### `GET /api/v1/history?limit=50&offset=0`
Returns a list of `AnalysisSummarySchema` (lightweight, for list views).

### `GET /api/v1/history/{analysis_id}`
Returns the full `AnalysisResultSchema` for one analysis. `404` if not found.

### `DELETE /api/v1/history/{analysis_id}`
Deletes an analysis record. `404` if not found.

### `GET /api/v1/statistics`
Aggregate dashboard data.
```json
{
  "total_analyses": 12,
  "category_distribution": {"construction": 5, "road_traffic": 4, "human_activity": 3},
  "loudness_distribution": {"quiet": 2, "moderate": 4, "loud": 5, "very_loud": 1},
  "average_dbfs": -19.4,
  "average_duration_sec": 4.1
}
```

## Reports & Export

### `GET /api/v1/reports/{analysis_id}/pdf`
Streams a generated PDF report for the given analysis. `404` if the analysis doesn't exist.

### `GET /api/v1/export/csv`
Streams a CSV file containing all stored analyses (flattened summary columns).
